# Financial & Sales Analysis Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes sales and financial performance using key business metrics.

## KPIs
- Total Sales
- Total Profit
- Total Quantity
- Total Discount

## Analysis
- Sales by customer segment
- Sales by product sub-category
- Sales trends over time
- Regional sales distribution

## Tools Used
- Power BI
- DAX
- Excel Dataset
